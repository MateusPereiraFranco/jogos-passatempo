Sobre o Jogo das 5 Palavras

Teste sua intuição! No Jogo das 5 Palavras, cada jogador escolhe 5 palavras secretas, onde cada palavra deve ter relação com a anterior. O objetivo é adivinhar as palavras do seu oponente antes que ele adivinhe as suas. É um jogo de turno, onde cada palpite conta. Quem descobrir todas as palavras primeiro, vence!

Sobre o Jogo das Senhas

Um desafio de pura lógica. No Jogo das Senhas, cada jogador define uma senha secreta de 4 números. Os jogadores então se revezam tentando adivinhar a senha um do outro. A cada tentativa, você descobre quantos números acertou. Quem decifrar o código primeiro é o grande vencedor!
